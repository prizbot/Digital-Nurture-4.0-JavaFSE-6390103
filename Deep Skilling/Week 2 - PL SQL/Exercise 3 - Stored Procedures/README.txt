Exercise 3: Stored Procedures

This package contains SQL scripts for practicing stored procedures in MySQL.

Files:
1. createTables.sql  - Creates Accounts, Employees, and Transactions tables.
2. insertData.sql    - Inserts sample data for accounts and employees.
3. scenario01.sql    - ProcessMonthlyInterest procedure to add interest to savings accounts.
4. scenario02.sql    - UpdateEmployeeBonus procedure to apply a bonus to employees in a department.
5. scenario03.sql    - TransferFunds procedure with balance check, transfer, and transaction logging.

Usage:
- Run createTables.sql to create tables.
- Run insertData.sql to populate tables.
- Execute the scenario scripts in a MySQL environment.
