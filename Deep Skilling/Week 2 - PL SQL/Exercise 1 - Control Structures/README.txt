Exercise 1: Control Structures

This package contains SQL scripts for practicing control structures in PL/SQL.

Files:
1. createTables.sql  - Creates Customers and Loans tables.
2. insertData.sql    - Inserts sample customer and loan data.
3. scenario01.sql    - Applies 1% interest discount for senior customers (Age > 60).
4. scenario02.sql    - Promotes customers with balance > 10000 to VIP.
5. scenario03.sql    - Lists loans due within the next 30 days.

Usage:
- Run createTables.sql to create tables.
- Run insertData.sql to populate tables.
- Execute the scenario scripts in an Oracle SQL environment.
