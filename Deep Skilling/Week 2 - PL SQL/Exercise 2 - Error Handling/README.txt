Exercise 2: Error Handling

This package contains SQL scripts for practicing error handling in PL/SQL.

Files:
1. createTables.sql  - Creates Customers, Accounts, Employees, and ErrorLog tables.
2. insertData.sql    - Inserts sample customer, account, and employee data.
3. scenario01.sql    - SafeTransferFunds procedure with error handling for insufficient funds.
4. scenario02.sql    - UpdateSalary procedure with error handling for missing employees.
5. scenario03.sql    - AddNewCustomer procedure with handling for duplicate Customer IDs.

Usage:
- Run createTables.sql to create tables.
- Run insertData.sql to populate tables.
- Execute the scenario scripts in an Oracle SQL environment.
